# 🌻 RAG Router Agent (Bindu)

## 🚀 Overview

This agent implements a Retrieval-Augmented Generation (RAG) system with intelligent database routing.

## 🧠 Features

* Intent classification
* Dynamic database selection
* Context retrieval
* LLM-based answer generation
* Structured response output

## ⚙️ How it works

User Query → Intent Detection → DB Routing → Retrieval → LLM Response

## 🔍 Example

Query: "What is GST?"

Output:

* Intent: finance
* DB Used: finance.txt
* Answer generated from context

## 💡 Why this matters

This agent demonstrates how Bindu agents can:

* Understand intent
* Route tasks intelligently
* Serve as coordination layers in multi-agent systems

## 🔥 Future Scope

* Multi-agent communication via A2A
* Vector database integration
* Confidence-based hybrid routing
